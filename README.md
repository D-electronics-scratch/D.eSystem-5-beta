D.eSystem 5+ Beta is a preview of the new modular architecture of D.eSystem.
This version focuses on a cleaner structure, separated modules, and a more scalable codebase.
The goal of 5+ is to replace the old monolithic “mode‑based” system from D.eSystem 5 with a real app‑based framework.

What’s new in D.eSystem 5+
A modular architecture with separate app functions

A command dispatcher instead of large if‑chains

A clean Home → App → Return flow

A more readable and maintainable codebase

Easier expansion for future apps (clock, notes, system info, etc.)

What’s included in this beta
Calculator app

News app

Basic shell structure

Home‑screen and command routing

Purpose of this beta
This beta is not a full OS.
It is a technical demonstration of the new architecture that will be used in future D.eSystem versions.
